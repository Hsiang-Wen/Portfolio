# The Worst Day

**Format**: Theatre play (20 min)  
**Role**: Sound Designer  
**Tools**: QLab, Logic Pro

## Description
This project was a collaborative theatre piece with live actors and digital media. I was in charge of sound cue planning, real-time playback coordination, and pre-production sound editing.

## Key Contributions
- Designed emotional transitions through ambient noise and musical underscoring
- Created cue list and triggered via QLab
- Worked closely with director on timing and texture
